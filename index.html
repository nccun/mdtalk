Mdtalk

一个简单的单对单会话聊天插件

Download this project as a .zip file Download this project as a tar.gz file

一个简单的单对单会话插件，基于jQuery

支持
  • 2种快捷键快捷键发送消息,Enter和Ctrl+Enter，并使用localStorage保存
  • 窗口可拖动，窗口位置用localStorage保存
  • 导入会话历史
  • 查看过去消息记录
